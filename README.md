# Тестовое задание — вёрстка арбитражного лендинга

## Описание

Проект создан в рамках тестового задания на позицию junior-верстальщика в арбитражной тематике.

Цель проекта:
- скопировать указанный лендинг
- привести проект к понятной локальной структуре
- удалить лишние сторонние скрипты
- внести требуемые правки в логику и поведение страницы
